# 第十一步：配置 COS 存储

## 📋 存储桶信息

- **存储桶名称**：`yang0313-storage-1318057968`
- **所属地域**：上海（ap-shanghai）

## 🚀 快速操作步骤

### 1. 访问 COS 控制台

**地址**：https://console.cloud.tencent.com/cos

### 2. 进入存储桶

1. 找到存储桶：`yang0313-storage-1318057968`
2. 点击存储桶名称进入详情

### 3. 配置访问权限

1. 进入「**基础配置**」→「**访问权限**」
2. 选择「**私有读写**」（推荐）或「**公有读私有写**」

### 4. 配置跨域规则（重要）

1. 进入「**安全管理**」→「**跨域访问 CORS**」
2. 点击「**添加规则**」
3. 配置：

```
来源 Origin: *
操作方法: GET, POST, PUT, DELETE, HEAD
Allow-Headers: *
Expose-Headers: ETag, x-cos-request-id
最大存活时间: 3600
```

4. 点击「**确定**」保存

### 5. 配置 CDN 加速（可选）

如果需要 CDN 加速：
1. 进入「**域名管理**」
2. 开启「**默认 CDN 加速域名**」

- [ ] 跨域规则已配置
