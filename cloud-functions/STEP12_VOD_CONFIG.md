# 第十二步：配置 VOD 云点播

## 🚀 快速操作步骤

### 1. 访问 VOD 控制台

**地址**：https://console.cloud.tencent.com/vod

### 2. 开通服务

1. 首次使用需要开通云点播服务
2. 选择计费方式：**按量计费**（适合初期）

### 3. 创建转码模板（可选）

如果需要自定义转码模板：
1. 进入「**视频处理**」→「**转码模板**」
2. 创建模板（流畅、标清、高清、超清）

### 4. 配置封面生成

1. 进入「**视频处理**」→「**截图模板**」
2. 创建封面模板：
   - 截图类型：时间点截图
   - 时间点：1秒
   - 图片格式：JPG
   - 分辨率：1280x720

### 5. 配置事件通知（重要）

1. 进入「**回调配置**」→「**事件通知**」
2. 点击「**添加回调配置**」
3. 配置：

```
回调模式: 可靠回调
回调 URL: https://yang0313-7g4dqwd46c63d876.ap-shanghai.app.tcloudbase.com/api/upload/vod/callback
事件类型: 
  ✅ 视频上传完成
  ✅ 视频转码完成
  ✅ 视频审核完成
```

> **注意**：回调 URL 需要在 `upload` 云函数中实现 `/upload/vod/callback` 接口

### 6. 配置播放域名（可选，需要备案）

如果需要自定义播放域名：
1. 进入「**分发播放设置**」→「**域名管理**」
2. 添加播放域名
3. 配置 CNAME 解析
4. 配置 HTTPS 证书
