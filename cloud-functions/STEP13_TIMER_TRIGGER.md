# 第十三步：配置定时触发器

## 📋 需要配置的函数

- **taskCheck** - 检查 VOD 任务状态、更新订单

## 🚀 快速操作步骤

### 1. 访问云函数控制台

**地址**：https://console.cloud.tencent.com/tcb/scf?envId=yang0313-7g4dqwd46c63d876

### 2. 进入 taskCheck 函数

1. 找到 `taskCheck` 函数
2. 点击函数名称进入详情

### 3. 配置定时触发器

1. 点击「**触发管理**」标签
2. 点击「**创建触发器**」
3. 配置：

```
触发方式: 定时触发
触发周期: Cron 表达式
Cron 表达式: 0 */1 * * * * *
启用: 是
```

**Cron 表达式说明**：
- `0 */1 * * * * *` - 每分钟执行一次
- `0 0 */1 * * * *` - 每小时执行一次
- `0 0 0 * * * *` - 每天执行一次

### 4. 保存配置

点击「**提交**」保存触发器配置。

## 📝 Cron 表达式格式

```
秒 分 时 日 月 星期 年
*  *  *  *  *  *   *
```

**常用表达式**：
- `0 */1 * * * * *` - 每分钟
- `0 0 */1 * * * *` - 每小时
- `0 0 0 * * * *` - 每天 0 点
- `0 0 12 * * * *` - 每天 12 点

## ✅ 验证

完成配置后，检查：
- [ ] 定时触发器已创建
- [ ] 触发器状态为「启用」
- [ ] 查看函数日志，确认定时执行
