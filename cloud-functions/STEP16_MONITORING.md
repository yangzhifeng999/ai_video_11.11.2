# 第十六步：配置监控与告警

## 🚀 快速操作步骤

### 1. 访问云监控控制台

**地址**：https://console.cloud.tencent.com/monitor

### 2. 创建告警策略

1. 点击「**告警管理**」→「**告警策略**」
2. 点击「**新建**」
3. 选择「**云函数**」作为策略类型

### 3. 配置告警规则

#### 规则 1：函数错误率告警

```
策略名称: 云函数错误率过高
监控指标: 错误率
告警条件: > 5%
持续周期: 2个周期
```

#### 规则 2：函数执行时间告警

```
策略名称: 云函数执行时间过长
监控指标: 执行时间
告警条件: > 5000ms
持续周期: 2个周期
```

### 4. 配置通知方式

1. 添加接收邮箱或手机号
2. 设置通知频率
3. 保存配置

### 5. 绑定告警对象

1. 选择需要监控的函数：
   - auth
   - user
   - video
   - order
   - upload
   - payment
   - admin
2. 保存配置
